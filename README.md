# NewsDigest Agent

Topic-Aware News Summarizer with Sentiment Analysis. 

## Problem 
Reading full news articles takes time.

## Solution
This AI agent fetches news based on topics, summarizes them into 3 sentences, and performs sentiment analysis.

## Features
- Topic-based news search
- Article summarization
- Sentiment detection
- Structured news digest

## Architecture
User → Web UI → Agent Controller → News API → Summarizer → Sentiment Analyzer → Output Digest
